# MailTriage Configuration
# All classification rules in one file

# ============================================
# ACTION KEYWORDS
# Emails containing these ‚Üí Action note
# ============================================

[ACTION_KEYWORDS]
swissport, aspire, lounge, airport lounge, priority pass, sbp, millar, Jagtar, David, fmorier, franbort

# ============================================
# ACTION CRITERIA
# Personal emails requiring response
# ============================================

[ACTION]
- Personal email from known contact with firstname.lastname@company format
- Subject contains "Re:" or "RE:" indicating ongoing conversation
- Email from @ehl.ch domain (school/work)
- Email asks a question or expects a response
- Contains SBP project keywords (swissport, millar, lounge, etc.)
- Written by a real person, not automated system

# ============================================
# INFORMATION CRITERIA
# Status updates and quality news worth noting
# ============================================

[INFORMATION]
- Confirmation of something YOU initiated (job application, order, form submission)
- Order confirmation or purchase receipt you want to track
- Refund or return status updates
- Course completion or certificate confirmation
- Quality news from: Reuters, L'Agefi, Bloomberg
- Transactional updates about YOUR actions (not marketing)

# ============================================
# IGNORE CRITERIA
# Everything else - do not save
# ============================================

[IGNORE]
- ALL emails from @linkedin.com (any subdomain)
- Emails from @marketing.*.com addresses
- Subject with emojis + discount/promo/code keywords
- Requests to rate, review, or give feedback on service
- Survey or satisfaction questionnaires
- Flight reminders ("your departure is soon") - you know your schedule
- App feature announcements or newsletters
- Bank security tips or generic newsletters
- System notifications (login links, "you used X service")
- Promotional emails, sales, limited time offers
- Connection requests, social media notifications

# ============================================
# DOMAIN RULES (strict)
# ============================================

[DOMAINS]
IGNORE: linkedin.com, marketing.easyjet.com, operational.easyjet.com
INFORMATION: thomsonreuters.com, agefi.com
ACTION: ehl.ch

# ============================================
# RULES
# ============================================

[RULES]
- When in doubt, choose IGNORE (60% of emails should be ignored)
- ALL LinkedIn emails are IGNORE, no exceptions
- Order confirmations are INFORMATION only if from the purchase, not marketing
- "Re:" or "RE:" from a real person = likely ACTION
- Promotional emails with emojis = IGNORE
- If they ask you to rate/review = IGNORE
